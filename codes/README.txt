LoiExpon, LoiGauss, LoiGamma sont des trois classes qui cherchent à obtenir la fonction de densité, la fonction de répartition et la fonction de la dérivée de densité des lois correspondantes.

les codes "asa147" est extraites à partir l'internet pour calculer la fonction de répartition de la loi gamma.

la classe "algo" vise à implémenter les deux méthodes.

"main.cpp" contient les tests.

Pour executer les codes, vous pourraient:
1. écrire ".\main" directement dans le terminal, qui va executer le programme "main" qui est déjà générés par nous.
2. si vous souhaitez modifier les codes, vous devez le re-compliler avec la commande "make", qui va réaliser ce qui est dans "Makefile", qui est déjà bien écrit. 

Sur Windows, il faut setup "mingw-make" d'abord si vous ne l'avez pas. 

Une fois que "make" est setuppé, vous pouvez recomplier et ré-exécuter.
